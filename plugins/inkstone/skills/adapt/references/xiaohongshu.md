# 小红书适配规范

adapt 适配小红书（Xiaohongshu）的完整规范。本文件定义首图标题、正文节奏、emoji+话题标签、钩子。

## 1. 平台调性

小红书是短图文生活分享社区，读者快速滑动浏览。调性：**短平快、口语化、有用或有趣**。强调「干货感」「真实感」「获得感」。可以是经验分享、避坑指南、好物种草、观点短打。篇幅通常 300-800 字，远短于公众号和知乎。

## 2. 首图标题

小红书是「图+文」笔记，首图和标题共同决定点击。adapt 产出的是文案，首图需用户另行制作，但要给标题文案建议：

- **标题要钩人**：数字、痛点、反差、身份标签最有效。
  - 数字型：「3 个让我存下钱的反直觉习惯」
  - 痛点型：「月薪 8k 也能存钱｜别再被这 3 个误区坑了」
  - 反差型：「读了几十年书才发现，我一直在无效阅读」
  - 身份型：「给 30 岁还在迷茫的你｜这章书救了我」
- **善用竖线分隔**：小红书标题常用「｜」分隔主标题和副信息，信息密度高。
- **字数**：20 字内为宜，关键信息前置（信息流只显示前几个字）。
- 拟 3-5 个候选标题供用户选。

> 注：首图文字通常与标题呼应，可提示用户把标题核心词放首图。adapt 不生成图片，仅给文案建议。

## 3. 正文节奏

小红书正文讲究「快节奏、强获得感」：

```
开头钩子（1-2 句）
  → 一句话点中痛点或抛出利益点，让读者觉得「这篇对我有用」

主体（分点呈现）
  → 用数字分点（1️⃣2️⃣3️⃣ 或 1. 2. 3.），每点一个干货/观点
  → 每点控制在 2-3 句，短平快
  → 把 source.md 核心论据拆成可独立消费的点

结尾（互动 + 价值）
  → 一句价值总结或号召
  → 可加互动引导（「你中过几条？评论区聊聊」）
```

要点：

- **分点**：小红书读者爱分点，清晰好扫。3-5 点为宜，太多则长。
- **每点有获得感**：不是空泛论述，每点落到一个具体可用的认知/动作。
- **短句**：句子要短，口语化，手机屏一屏能看几行。
- **避免长段落**：超过 3 行的段落就拆。

## 4. emoji + 话题标签

小红书的视觉语言，用好提升亲和力和曝光：

### emoji

- **适度使用**：每段开头或关键点配 1 个 emoji，强化视觉节奏。
- **选与内容相关的**：理财用 💰📈、成长用 🌱📚、避坑用 ⚠️❌。不堆砌无关 emoji。
- **克制**：不要每句都加，3-5 个 emoji 贯穿全文即可，过多显廉价。

### 话题标签

- **文末加标签**：3-8 个 `#话题` 标签，放文末。
- **组合策略**：1-2 个大流量泛标签 + 2-3 个精准垂直标签 + 1 个内容相关标签。
  - 泛标签：`#个人成长` `#理财` `#读书分享`
  - 垂直标签：`#存钱方法` `#投资入门` `#书单推荐`
  - 相关标签：贴合本篇具体内容的
- 标签从 source.md 核心观点和受众推断，给候选供用户调整。

## 5. 钩子

小红书靠钩子留住滑动的读者：

- **开头钩子**：第一句就要抓人——痛点、利益点、反差、数字。
- **分点钩子**：每个分点的小标题本身是钩子（「误区一：以为收入低就存不下钱」而非「第一点」）。
- **结尾钩子**：留一个互动或悬念，引导评论收藏（评论和收藏是小红书重要互动指标）。

source.md 的可引用金句是天然钩子素材——短、有冲击力、可独立成立。

## 6. 语感

小红书语感最口语、最亲昵：

- 像在跟闺蜜/朋友安利一个发现，有热情、有真实感。
- 第一人称口吻（「我发现…」「真的会被这句戳到」），保留 source.md 读者自己的表达。
- 可以有情绪和感叹，但不过度（满篇「绝绝子」「yyds」反而廉价）。
- 避免 AI 腔：不用机械排比、不用书面八股、不用「在当今社会」式开场。
- 「干货感」优先：读者刷小红书要的是「有用」，情绪服务于有用，不空泛抒情。

## 7. 从 source.md 取材

| source.md 字段 | 小红书用途 |
|----------------|-----------|
| 核心观点 | 标题影子 + 开头钩子 + 结尾价值总结 |
| 关键论据素材 | 拆成分点干货，每点一个可独立消费的认知 |
| 可引用金句 | 开头钩子、分点锚定、结尾点睛 |
| 受众 | 决定痛点切入和举例贴近度（如学生党/职场新人） |
| 备注 | 守禁忌；「待深入」点不放进小红书（短文承载不了复杂思辨） |

- [ ] 句子短、无长段落？

- [ ] source.md 备注的禁忌点未触碰？
